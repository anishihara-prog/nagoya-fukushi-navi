# 障害福祉支援ナビ なごや(試作版)

名古屋市の障害者相談員の方が、福祉サービス・必要な手続き・ケースに応じた支援を検索できるツールの試作版です。
ビルドツール不要(npm install なし)の、HTML / CSS / 素のJavaScriptだけで作られています。

## フォルダ構成

```
nagoya-fukushi-navi/
├── index.html          ← アプリ本体(これをブラウザで開く)
├── css/style.css        ← デザイン
├── js/
│   ├── data-store.js     ← データの読み書きだけを担当する層(将来差し替え用)
│   └── app.js            ← 検索・ケース検索・登録編集のロジック
└── data/entries.json     ← サンプルデータ(福祉サービス・手続き 10件)
```

## VSCodeでの開発の始め方

1. このフォルダをVSCodeで開く(「フォルダーを開く」)
2. `index.html` を直接ダブルクリックでは**開かないでください**(`file://` で開くとデータの読み込みに失敗します)。
   代わりに、以下のどちらかの方法でローカルサーバーを立てて開いてください。
   - **VSCode拡張機能「Live Server」を入れている場合**: `index.html` を右クリック →「Open with Live Server」
   - **拡張機能を入れたくない場合**(ターミナルで):
     ```
     cd nagoya-fukushi-navi
     python3 -m http.server 5500
     ```
     ブラウザで `http://localhost:5500` を開く
3. ブラウザが開いたら、タブを切り替えて動作を確認してください。
   - 🔍 検索する: キーワード・分類で絞り込み
   - 🧭 ケースから探す: 障害種別・年代・困りごとのチップを選ぶと、当てはまる支援が表示される
   - 🗂 登録・編集: 情報の追加・編集・削除、JSONの書き出し/読み込み

## 今のデータの保存場所について(重要)

今の実装では、登録・編集した内容は**自分のブラウザのlocalStorageだけ**に保存されます。
つまり今の段階では「自分のPC・自分のブラウザの中だけ」のデータで、他の相談員とは自動的には共有されません。

チームでデータを育てていく場合は、今のところ次の運用がおすすめです。

1. 担当者が「登録・編集」タブでデータを追加・修正する
2. 「⭳ JSONを書き出す」でファイルをダウンロードする
3. ダウンロードした内容で `data/entries.json` を上書きする
4. Gitにコミットして、他のメンバーは最新の `data/entries.json` を `git pull` する
   (他のメンバーが反映するには、ブラウザの開発者ツールでlocalStorageを一度クリアしてから開き直す必要があります)

これは**あくまで初期段階のやり方**です。本当に名古屋市の相談員全体でリアルタイムに共有するには、
下の「次の段階」が必要になります。

## 次の段階(本格的に全市で共有するなら)

このままでは「同時に複数人が編集する」「常に最新のデータを全員が見られる」という状態は作れません。
本格的に展開するには、以下のような小さなバックエンドが必要になります。

- 簡単なAPIサーバー(Node.js + Express など)を1つ用意する
- データはサーバー側のデータベース(SQLiteなど、小規模なら十分)に保存する
- `js/data-store.js` の `loadEntries` / `saveEntries` の中身を、
  localStorageの代わりに `fetch('/api/entries')` を呼ぶ処理に書き換える
  (ファイル冒頭にサンプルコードを書いています)
- `app.js` 側は変更不要です

その先、名古屋市の正式なツールとして全相談員に展開する場合は、データの安全な保管・アクセス権限・
アクセシビリティ対応などの観点から、市の情報システム部門との連携・正式な手続きを検討することをおすすめします。

## 個人情報の取り扱いについて(必ずお読みください)

このツールは「制度・サービスの一般的な情報」を検索するためのものです。
相談者個人を特定できる情報(氏名・住所・個別の相談記録など)は、**入力しないでください**。
特に将来「全相談員で共有」する形にした場合、入力した情報は他の利用者にも見える想定で設計されています。

## サンプルデータについて

`data/entries.json` に入っている10件は試作用のサンプルです。内容は一般的な公開情報を参考に作成していますが、
実際の運用前には、各区役所・基幹相談支援センター等で最新の正確な情報に必ず更新してください。
